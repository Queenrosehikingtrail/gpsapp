🏔️ Queen Rose Hiking App

Android Installation & Setup Guide

⚠️ Important: Why Apps Sometimes Disappear

Android automatically manages apps to save battery and storage. PWAs (web apps) like Queen Rose Hiking can be removed if not properly installed or configured. This guide ensures your app stays installed permanently.

1Proper Installation Method

Use Chrome Browser Only (not Samsung Internet, Firefox, or other browsers)

  1. Open Chrome browser on your Android device
  2. Navigate to your Queen Rose Hiking app URL
  3. Look for the "Add to Home Screen" popup at the bottom
  4. If no popup appears, tap Chrome menu (⋮) → "Add to Home Screen"
  5. Tap "Add" when prompted
  6. The app should now appear in your app drawer (not just home screen)

✅ Verify Proper Installation

After installation, check:

  • App appears in Android app drawer (swipe up from home screen)
  • App opens in full screen (no browser address bar)
  • App appears in Settings → Apps list

2Configure Chrome Settings

Prevent Android from removing the app

  1. Go to Settings → Apps → Chrome
  2. Tap "Battery" → Select "Don't optimize"
  3. Go to Settings → Battery → Battery Optimization
  4. Find "Chrome" → Set to "Don't optimize"
  5. Ensure Chrome has sufficient storage space

📱 Device-Specific Settings

Samsung Devices:

  • Settings → Device Care → Battery → App Power Management
  • Add Chrome to "Never sleeping apps"
  • Disable "Put apps to sleep"

Huawei/Honor Devices:

  • Settings → Battery → App Launch → Chrome
  • Set to "Manual" and enable all options

Xiaomi/MIUI Devices:

  • Settings → Apps → Manage Apps → Chrome
  • Enable "Autostart"
  • Disable battery restrictions

OnePlus Devices:

  • Settings → Battery → Battery Optimization
  • Chrome → "Don't optimize"

3Regular Usage Pattern

Keep the app active to prevent removal

  • Use the app at least once per week
  • Always open from home screen icon (not browser)
  • Interact with app features (view trails, check GPS, etc.)
  • Use offline features to demonstrate app value to Android

✅ Setup Checklist

Complete this checklist to ensure maximum app persistence:

  • Installed app using Chrome browser
  • App appears in Android app drawer
  • App opens in full screen mode
  • Chrome battery optimization disabled
  • Device-specific settings configured
  • App used at least once this week
  • Sufficient storage space available

🔧 Troubleshooting: If App Still Disappears

  1. Reinstall using Chrome (not other browsers)
  2. Check all battery optimization settings
  3. Use app more frequently (daily for first week)
  4. Clear Chrome cache and reinstall
  5. Update Chrome to latest version
  6. Try Samsung Internet (Samsung devices only)

🎯 Expected Results

Following this guide should result in:

  • 70-80% persistence with proper setup
  • 90%+ persistence with regular usage
  • Reliable offline functionality
  • Native app-like experience
